# PHERUSA
### React 前端项目模版
(海神女，涅柔斯和多丽丝的五十个女儿之一)

### 安装

```
npm i / yarn

```

### 启动

```
npm start / yarn start
```

### 启动mock

```
npm run mock
```

### 打包

```
npm run build
```
